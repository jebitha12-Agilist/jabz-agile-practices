# Agile Maturity Matrix for Teams 📈

## 🎯 Purpose
Assess team agility health, set transformation goals, and monitor progress.  
This matrix helps Scrum Masters, Agile Coaches, and leadership align on maturity levels and drive continuous improvement.

---

## 📝 Instructions
1. **Rate Current Level** → Use scale 0–4:  
   - 0 = Impeded  
   - 1 = In Transition  
   - 2 = Sustainable  
   - 3 = Agile  
   - 4 = Ideal  

2. **Set Target Level** → Define desired maturity for each area.  
3. **Add Comments** → Capture actions needed to reach goals.  
4. **Review Radar Chart** → Visualize maturity progress (blue line = current, outer rim = ideal).  
5. **Evaluate Quarterly** → Track improvements and adjust transformation roadmap.

---

## 📊 Key Areas
- **Team Dynamics** → Agile mindset, collaboration, morale.  
- **Teamwork & Self‑Organization** → Trust, integrity, shared responsibility.  
- **Sustainable Pace** → Avoid burnout, maintain consistent delivery.  
- **Working Agreement** → Documented, visible, and followed.  
- **Environment** → Team size, continuity, cross‑functional skills, collocation.  
- **Impediment Management** → Raising and resolving blockers as cultural norm.  
- **Product Practices** → Shippability, backlog grooming, INVEST, DoR/DoD.  
- **Agile Process Mechanics** → Standups, retrospectives, reviews, estimation.  
- **Engineering Practices** → Testing, automation, CI/CD, refactoring.
